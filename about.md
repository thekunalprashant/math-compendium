# About Math Compendium

Personal notes documenting competition math techniques. Built for quick reference and rapid review before contests.

## Contribute

Want to add content? Here's how:

1. **Fork** the repository on GitHub
2. **Add your topic** in `/topics/your-topic-name/index.md`
3. **Submit a Pull Request**

### Topic Format

```yaml
---
layout: page
title: Your Topic Name
description: Brief description of what this covers
date: 2025-01-20  # For sorting
popular: true    # Optional: mark as popular
permalink: /topics/your-topic-name/
---

Guidelines
Use clear examples with step-by-step solutions
Include practice problems with solutions
Keep explanations concise (competition-focused)
Use LaTeX for math: $x^2 + y^2 = z^2$
---
```
