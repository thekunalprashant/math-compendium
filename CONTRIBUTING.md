# Contributing to StudyBase

Thank you for your interest in contributing! This project follows a structured academic format to ensure clarity and professional aesthetics.

## How to Contribute

1. **Fork the Project:** Create your own branch by forking the repository.
2. **Add Your Topic:** Topics reside in subject-specific directories:
   - Path: `/subjects/[subject-name]/[topic-name]/index.md`
3. **Use the Template:** Follow the standardized frontmatter:
   ```yaml
   ---
   layout: page
   title: Your Topic Name
   description: A concise summary of the method
   category: Algebraic Manipulation
   popular: true
   ---
   ```
4. **Submit a PR:** Open a Pull Request for review.

## Guidelines

- **LaTeX:** Use LaTeX for all mathematical expressions (e.g., $x^2$).
- **Modularity:** Extract reusable logic into `_includes/` where appropriate.
- **Grayscale Aesthetic:** Stick to the project's minimalist color scheme.
